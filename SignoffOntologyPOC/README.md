# Signoff Ontology POC

This is a Mock Dash Application to demonstrate the Signoff Platform Ontology.

## Prerequisites

- Python 3.8+
- pip

## Installation

1. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Running the App

1. Run the application:
   ```bash
   python app.py
   ```
2. Open your web browser and navigate to:
   `http://127.0.0.1:8050/`

## Features

- **Ontology Map**: Visualizes the relationships between Signoff Objects (Product, Revision, Block, etc.).
- **Digital Twin Explorer**: Allows you to drill down into specific Products and Revisions to see the status of Tasks and Jobs.
- **AI Agent Demo**: Simulates how an AI Agent would use the Ontology to answer questions like "Status Check" or "Waiver Analysis".
